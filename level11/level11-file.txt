Другие компании быстро оценили потенциальные возможности Docker. В сен-
тябре 2013 года основным партнером становится компания Red Hat, которая нача-
ла использовать Docker в качестве движка своего облачного продукта OpenShift.
Вскоре после этого Google, Amazon и DigitalOcean обеспечили поддержку Docker
в своих облачных сервисах, а некоторые стартапы стали специализироваться на
создании Docker-хостов, например компания StackDock. В октябре 2014 года
корпорация Microsoft объявила о полной поддержке Docker в будущих версиях
Windows Server, что свидетельствовало о существенном изменении позициониро-
вания компании, ранее знаменитой своим массивным, «неповоротливым» корпо-
ративным программным обеспечением.
